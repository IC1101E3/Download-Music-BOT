Download-Music-BOT By IC1101E3

It is a simple bot that downloads audio from YouTube videos on Telegram.

Importance!

There are still some things that need improvement! Although it can be run successfully.

Run

Clone this repository: `git clone https://github.com/IC1101E3/Download-Music-BOT'
Download the dependencies: `pip install -r requirements.txt`
Download the FFmpeg library
